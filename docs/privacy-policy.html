Privacy Policy

We don't store your data, period. This application's only purpse is to convert a Google document to a CSV format. To serve this purpose two Google API scopes are required

1: https://www.googleapis.com/auth/documents.currentonly

This scope allows access to the current document to convert it into a CSV format<./p>

2: https://www.googleapis.com/auth/script.container.ui

This scope allows access to create a new modal window that allows you to download the CSV file to your computer.
